## Красивый DMG (фон + позиции иконок)

`build.sh` умеет собирать “красивый” DMG, если вы положите фон сюда:

- `assets/dmg/background.png`

### Рекомендованные параметры фона

- **Размер**: 600×400 px (под окно Finder)
- **Стиль**: чёрный/тёмно‑серый фон + оранжевый градиент/акцент + стрелка между иконками
- **Формат**: PNG

### Шаблон

В этой папке есть текстовый шаблон `background-template.svg`. Его можно открыть в Figma/Illustrator/Inkscape
и экспортировать как `background.png`.

### Как выглядит раскладка

- `WeDPI.app` слева
- `Applications` справа
- Иконки по центру окна; фон задаётся как background picture
