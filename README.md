WeDPI — приложение в menubar для запуска SpoofDPI и обхода DPI на macOS.

## Скачать и установить

- **Releases**: откройте страницу релизов вашего репозитория и скачайте `WeDPI.dmg`.
- **Установка**: перетащите `WeDPI.app` в `/Applications`.
- **Первый запуск**: если macOS блокирует запуск — сделайте **правый клик → Открыть**.

## Требуется SpoofDPI

WeDPI использует внешний бинарник `spoofdpi`.

- **Установка через Homebrew**:
  - `brew install spoofdpi`
- **Или Go**:
  - `go install github.com/xvzc/SpoofDPI/cmd/spoofdpi@latest`

Если SpoofDPI не найден, приложение покажет подсказку в интерфейсе.

## Использование

- **Подключить**: запускает `spoofdpi` на `127.0.0.1:<порт>` и включает системный прокси.
- **Отключить**: останавливает `spoofdpi` и выключает прокси.
- **Настройки**: порт, автозапуск, логи.

## Обновления

В приложении есть проверка обновлений через GitHub Releases:

- **Настройки → О программе → Проверить обновления**

Если доступен новый релиз и в нём есть `*.dmg`, появится кнопка скачать DMG.

## Частые проблемы

- **“Приложение повреждено” / “Нельзя открыть”**:
  - Системные настройки → Конфиденциальность и безопасность → **Открыть в любом случае**
  - или:
    - `xattr -dr com.apple.quarantine /Applications/WeDPI.app`
- **Не работает Discord/Instagram**:
  - WeDPI может запросить права для BPF (появится системный запрос пароля).

## Приватность

WeDPI не собирает и не отправляет ваши данные. Работает локально: запускает `spoofdpi` и меняет настройки прокси в системе.
